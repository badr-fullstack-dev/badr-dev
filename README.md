# Hi, I'm Badr 👋

I'm a full-stack web developer based in Paris. I build websites, web apps, and AI-powered tools, and I work a lot with AI in my day-to-day workflow — both as a tool to build faster, and as something I integrate into the products themselves.

A big part of what I do these days is AI automation: connecting LLMs to real workflows so people stop doing repetitive tasks by hand. Things like chatbots, internal tools, content generators, lead handlers. I also do prompt engineering, and I genuinely enjoy figuring out how to make AI features feel reliable instead of just impressive in a demo.

I'm currently open to remote full-stack roles and freelance projects. I speak French, English, and Arabic.

## What I work with

* **Frontend:** React, TypeScript, JavaScript, HTML, CSS, Tailwind
* **Backend:** Node.js, Express, Django, REST APIs, MongoDB
* **AI:** OpenAI, Claude, prompt engineering, LLM integrations
* **Other:** Git, Vite, Postman, Vercel, Render
