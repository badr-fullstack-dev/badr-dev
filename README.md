## About me

I'm a full-stack web developer based in Paris. I build websites, web apps, and AI-powered tools, and I work a lot with AI in my day-to-day workflow — both as a tool to build faster, and as something I integrate into the products themselves.

A big part of what I do these days is AI automation: connecting LLMs to real workflows so people stop doing repetitive tasks by hand. Things like chatbots, internal tools, content generators, lead handlers. I also do prompt engineering, and I genuinely enjoy figuring out how to make AI features feel reliable instead of just impressive in a demo.

I'm currently open to remote full-stack roles and freelance projects. I speak French, English, and Arabic.

## A few projects I've worked on

**Jarvis Desktop AI Assistant** ([repo](https://github.com/badr-fullstack-dev/Jarvis-Desktop-AI-Assistant))
This is my personal project and the one I'm most excited about. A desktop AI assistant that takes voice commands, automates tasks on your machine, and uses LLMs to actually have a conversation with you. I'm using it to figure out where AI assistants are genuinely useful versus where they're just a novelty.

**UMOD** ([umod.fr](https://umod.fr))
Client work for a French e-commerce business. I built them a custom calculator tool ([frontend](https://github.com/badr-fullstack-dev/Umod-Calculator-Aides-frontend), [backend](https://github.com/badr-fullstack-dev/Umod-Calculator-Aides-backend)), a chatbot ([frontend](https://github.com/badr-fullstack-dev/Umod-Chat-Bot-frontend), [backend](https://github.com/badr-fullstack-dev/Umod-Chat-Bot-backend)), and their [landing page](https://github.com/badr-fullstack-dev/umod-page-acceuil2). Took it from design to production.

**Haltea & Cogelas** ([Haltea](https://github.com/badr-fullstack-dev/haltea-website), [Cogelas](https://github.com/badr-fullstack-dev/cogelas-website))
Two more client websites. Both businesses, both shipped, both running.

**E-commerce platform** ([frontend](https://github.com/badr-fullstack-dev/ecommerce-website-frontend), [backend](https://github.com/badr-fullstack-dev/ecommerce-website-backend))
A personal full-stack project. Product catalog, cart, checkout, the whole thing. I built it because I wanted to do every layer myself and see it work together.
